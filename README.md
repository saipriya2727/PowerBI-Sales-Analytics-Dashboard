# Chocolate Sales Analytics Dashboard | Power BI

## Project Overview

This project is an interactive Power BI dashboard developed to analyze chocolate sales performance across different regions, products, and salespersons. The dashboard provides business insights through KPI tracking, sales trend analysis, profit analysis, and geographic distribution.

## Objectives

* Monitor sales performance and profitability.
* Compare Current Year (CY) vs Previous Year (PY) sales.
* Analyze top-performing products and salespersons.
* Track shipment distribution and regional sales trends.
* Support data-driven business decisions.

## Tools & Technologies

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Excel Dataset

## Dashboard Features

### 1. Executive Dashboard

* Total Sales Amount
* Total Boxes Sold
* Shipment Count
* Total Profit
* Profit Percentage
* Top Products Analysis
* Top Salespersons Analysis

### 2. Sales Performance Analysis

* Monthly Sales Trends
* Year-over-Year (YoY) Comparison
* Geographic Sales Distribution

### 3. Profit Analysis

* Salesperson-wise Profit Analysis
* Profit Percentage Comparison
* Top Profit Contributors

## Key Insights

* Identified the highest revenue-generating products.
* Tracked regional sales contributions.
* Compared sales performance across different periods.
* Evaluated salesperson profitability and efficiency.
